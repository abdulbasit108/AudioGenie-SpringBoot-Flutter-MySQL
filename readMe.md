# Audio Genie

**Audio Genie** is a full-stack application designed to extract key information from audio files, such as entities, timestamps, and more. This project is divided into two parts: the **frontend** built with Flutter, and the **backend** built with Spring Boot.

## Features

- **Audio Processing**: Extracts and displays meaningful data from audio files.
- **Entity Recognition**: Identifies and highlights entities such as dates, names, locations, and more.
- **User Authentication**: Secure login and registration.
- **History Management**: Users can view and manage their transcription history.

## Tech Stack

### Backend
- **Java** with **Spring Boot**
  - REST API
  - JWT Authentication
  - MySQL Database for data persistence
  - Lombok for cleaner code

### Frontend
- **Flutter**
  - Responsive UI
  - Material Design
  - Integration with backend via REST API
